Slider

Slider or range input is used to specify a particular range, out of which a user cannot select a value. Inputs are means of getting data from user. So, in a sense, it can be said inputs are also a way for user to interact with the interface of the website. The slider input is used when the exact value of input is not the target from the user, rather, it targets a range of values.

No step value

These are sliders that do not have any step value, and the transition is smooth.

Minimum step value sliders

These are sliders that have a minimum step value for every stride of the slide.
