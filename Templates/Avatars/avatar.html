Avatars

An avatar is a graphical representation of a user or the user's character or persona. Avatars usually have matching height and width, and can be used as the profile picture or display picture.

Square Avatars

Basic square avatars. These are default so you do not need to explicitly define a class for the shape. Just define the margins and avatar size. for square avatars.

Soft Bordered Avatars

Slightly rounded avatar borders. Use class softAvatar for square avatars.

Round Avatars

These are circular avatars. Use class roundAvatar for round avatars.

Letter Avatars

When an image is not available, user can also use a letter as his avatar. Use class roundAvatar for round avatars.
