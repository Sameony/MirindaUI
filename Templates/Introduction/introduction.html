Introduction

Mirinda UI is a component library, which contains the classes one usually needs to style and create elements. When creating a website, major part of the time goes into adjusting the styling of the webpages. With the help of Mirinda UI's available classes, you can easily create different components fully styled in a few minutes, so that you can focus on other tasks, such as adding functionality and wrap up quickly.
You can just import this link into your css file and you are ready to start:
