Buttons

Solid Buttons

Buttons are used mainly as functionality triggers, that is, for communicating with the user. Buttons can be of many types, but mostly, they are either clickable links or just buttons. These are important components as they are used throughout a website, from header and navigation, cards, links to footer and social connect links. 
            
        <button class="btn btn-primary">Primary Button</button>
        <button class="btn btn-secondary">Secondary Button</button>
        <button class="btn btn-caution">Caution Button</button>
        <button class="btn btn-success">Success Button</button>

Outline Buttons

Outline Buttons are used to provide a bit "Lighter" look for the buttons. They usually dont have filled colors in the background. The outline style buttons can be used for various purposes, such as the secondary action complementing the primary action or If you have many buttons, then it may help in reducing the heavy color traffic. 
            
        <button class="btn btn-outline-primary">Primary Button</button>
        <button class="btn btn-outline-secondary">Secondary Button</button>
        <button class="btn btn-outline-caution">Caution Button</button>
        <button class="btn btn-outline-success">Success Button</button>

Link Buttons

Link buttons are not technically buttons, but rather hyperlinks to same or another website or webpage.
Squared Link Borderless Link Dashed Link 
            
        <a class="btn-link btn-outline-primary mg-y-sm fw-xl" href="target_Link_Here">Squared Link</a>
        <a class="btn-link btn-link-noBorder mg-y-sm fw-xxxl" href="target_Link_Here">Borderless Link</a>
        <a class="btn-link btn-link-dot mg-y-sm fw-xl" href="target_Link_Here">Dashed Link</a>

Buttons w/ Icons

Normal buttons but they have icons along with them, that signify something embedded in them. 
            
        <button class="btn btn-primary mg-y-sm"><i class="fas fa-shopping-cart mg-r-sm"></i>Add to Cart</button>
        <button class="btn btn-caution mg-y-sm"><i class="fas fa-trash mg-r-sm"></i>Delete</button>
        <button class="btn btn-share mg-y-sm"><i class="fas fa-share-alt mg-r-sm"></i>Share</button>

Floating Action Button

A floating action button is a button that has a fixed place on a webpage/viewport of the user, stacked over other elements of the webpage, and performs it's designated task, which is usually something like scroll to top, open box of menu or actions etc. 
            
        <button class="btn btn-primary pd-sm fs-lg btn-floating-action mg-y-sm shadows"><i class="fas fa-chevron-up mg-r-sm"></i></button>
        <button class="btn btn-outline-primary pd-sm fs-lg btn-floating-action mg-y-sm shadows"><i class="fas fa-plus mg-r-sm"></i></button>
