Inputs

Inputs are means of getting data from user. So, in a sense, it can be said inputs are also a way for user to interact with the interface of the website.

Text Input fields

These are the basic input areas, and the basic necessity to get user to input custom data and not select an answer from predefined options 
                
        <div class="contentInput">
            <div class="boxInput mg-y-sm">
                <label for="name" class="boxLabel">Name</label>
                <input type="text" id="name" class="itemInput fs-l">
            </div>
            <div class="boxInput">
                <label for="password" class="boxLabel">Password</label>
                <input type="password" id="password" class="itemInput fs-l">
            </div>
        </div>

Validated Inputs

These are styled according to response to a validation function, whether validation was a success or a failure.

This username is available.

The password you entered is invalid. 
                    
        <div class="contentInput">
            <div class="boxInput mg-y-sm">
                <label for="name" class="boxLabel">Name</label>
                <input type="text" id="name" class="itemInput validateSuccess fs-l"><br>
                <span class="successMessage">
                <i class="fas fa-check-circle"></i>This username is available.
                </span>
            </div>
            <div class="boxInput">
                <label for="password" class="boxLabel">Password</label>
                <input type="password" id="password" class="itemInput validateFailure fs-l"><br>
                <span class="cautionMessage">
                <i class="fas fa-times-circle"></i>The password you entered is invalid.
                </span>
            </div>
        </div>

Radio And CheckBox

These are inputs which are obtained from user from a specific pre-defined set of options.

Guess what sound does the cat make?

I would like to buy the following items:

            
        <div class="contentInput">
            <div class="boxInput mg-y-sm">
                <input class="mg-x-sm" type="radio" name="voice" id="cat">
                <label for="cat">Meow</label>
                <input class="mg-x-sm" type="radio" name="voice" id="dog">
                <label for="dog">Bark</label>
                <input class="mg-x-sm" type="radio" name="voice" id="frog">
                <label for="frog">Croak</label>
            </div>
            <div class="boxInput mg-y-sm">
                <input type="checkbox" name="shopping" id="choco">
